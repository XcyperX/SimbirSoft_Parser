Для запуска приложения необходимо:
•	Скачать и установить JVM https://www.java.com/ru/download/
•	Запустить из проекта jar файл «Parser-0.0.1-SNAPSHOT.jar»
•	В браузере перейти по ссылке http://localhost:8080/

После запуска приложения и перехода по ссылке http://localhost:8080/ загрузиться интерфейс, с возможностью
распарсить страницу, и просмотреть историю предыдущих страниц.

Для того чтобы распарсить страницу, необходимо:
•	Ввести ссылку в единственное input поле
•	Нажать на кнопку «Парсить»
•	Ждать пока появится уведомление «Готово!»
После этого информация о парсинге страницы появится в таблице.

В таблице есть возможность посмотреть результаты парсинга нажав на кнопку «Подробности», а также сохранить
результаты в формате pdf на жесткий диск, нажав на кнопку «Сохранить».
